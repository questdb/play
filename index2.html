There once was an engineer who dreamt of an open source time series database that could ingest data at a rate of millions rows per second, and support fast SQL queries with dedicated functionality for grouping and joining data across time.

That product is now known as QuestDB. Things have moved on since then, with users in industries such as finance, shipping, energy, crypto, IoT and many more.

Nowadays QuestDB very well known for performance, but we also want the product to be known for what it's often most loved for: It's ease of use.

We've set up this website, play.questdb.io to make it easy for anyone to try out QuestDB.

This one goes out to all the programmers and data scientists who want to try us out, but don't want to go through the hassle of installing software, setting up an instance and finding some data to play with.

We've bundled it all up in a single command that you can run in your terminal. We've got two options for you:

The first is python3 -c "import urllib.request as w;exec(w.urlopen('https://dl.questdb.io/play/run.py').read())" and launches through Python into a temporary directory (which we go to lengths to ensure is properly cleaned up after you're done).

The second is docker run -p 8888:8888 -p 8812:8812 -p 9009:9009 -p 9000:9000 questdb/play:1.0.0 and launches a docker container.

Either way, there'll be nothing left installed on your system after trying us out.

We've given you some data to play with too: It's grid energy usage and forecasts at 15 minute intervals. Just under half a million rows.

You'll be able to play with the data, try queries out and plot a few graphs.

Start playing! We hope you enjoy it.
